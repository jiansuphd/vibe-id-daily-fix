# Day 76: The 'Course Lifecycle' Agent

- **The Pointer:** Automate the complete course lifecycle: from blueprint to build to QA to launch to retirement — with agent-assisted handoffs at each gate.
- **The Details:** The lifecycle has predictable gates: design review, accessibility check, faculty sign-off, launch, mid-semester check, end-of-semester archive. An agent manages the checklist and escalations.
- **Action:**

```bash
gemini "Given this course status (stage: QA), generate the next 3 action items with owners, deadlines, and dependencies. Flag any blockers based on the current date."
```
