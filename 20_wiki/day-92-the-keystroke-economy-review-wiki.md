# Day 92: The 'Keystroke Economy' Review — Wiki

## Core Concept

Run the quarterly review of the keystroke economy: which automations are paying off, which tasks have crept back to manual, and where the next 10x efficiency gain lives. Automation ROI decays if not maintained. New tasks appear, existing automations break, and manual processes re-emerge. The quarterly review resets the baseline and identifies the next highest-value target.
