# Automated Rubric Calibration (The Fairness Audit) - Pedagogical Synthesis

*Expanded concepts from the Day 37 daily pointer.*

## Core Concept

"Rubric Drift" occurs when a rubric's grading criteria no longer directly measure the stated learning objective. The Fairness Audit uses triangulation - simultaneously feeding the Syllabus, Rubric, and a Sample Assignment to the AI - to expose misalignments before students are graded.

## The Triangulation Model

```
Syllabus (Objectives) ──┐
Rubric (Criteria)    ────┼──► AI Conflict Report ──► Revision Recommendations
Sample Assignment    ──┘
```

## Common Rubric Drift Patterns

| Drift Type | Example | Fix |
|---|---|---|
| Weight mismatch | 20% on "Analysis" but no analytical task in assignment | Add case analysis component |
| Bloom mismatch | Objective says "Evaluate" but rubric measures "List" | Raise rubric ceiling to synthesis/evaluation |
| Criteria vagueness | "Good writing" without measurable indicator | Replace with "Topic sentence + 2 supporting citations" |

## ID Application

- ** graduate courses:** High-stakes grading requires defensible, consistent rubrics
- **QM Standard 3:** Assessments must measure stated module/course objectives
- **Cross-instructor calibration:** Adjunct faculty grading consistency audit
