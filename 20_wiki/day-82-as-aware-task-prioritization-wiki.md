# Day 82: AS-Aware Task Prioritization (Dynamic) — Wiki

## Core Concept

Build a dynamic task prioritization system that re-ranks the day's work based on real-time energy and pain signals. Static task lists assume constant capacity. A dynamic system asks about today's AS status at 8 AM and re-ranks accordingly — protecting cognitive work for good days and deferring it on flare days.
