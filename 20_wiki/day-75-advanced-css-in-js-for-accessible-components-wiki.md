# Day 75: Advanced CSS-in-JS for Accessible Components — Wiki

## Core Concept

Generate accessible, theme-aware UI components for course shells that pass WCAG AA contrast checks automatically. Custom LMS CSS can accidentally break accessibility. An agent generates CSS that is WCAG-compliant by construction, with contrast ratios embedded as custom properties.
