# Day 68: The 'Institutional Memory' RAG — Wiki

## Core Concept

Build a Retrieval-Augmented Generation index over your entire vault so any prompt can draw on past decisions without manual context injection. RAG turns your vault into a queryable memory. Every decision log, SOP, and skill file becomes a retrievable fact. The agent answers 'what did we decide about X?' from the corpus, not from hallucination.
