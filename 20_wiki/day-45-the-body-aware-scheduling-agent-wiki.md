# Day 45: The 'Body-Aware' Scheduling Agent (AS Resilience) — Wiki

## Core Concept

Build a scheduling agent that respects AS (Ankylosing Spondylitis) flare cycles, fatigue windows, and ergonomic constraints. Your condition is a real workplace variable. An agent that models energy levels and blocks high-cognitive tasks during flare windows is not a luxury — it's an ADA accommodation strategy.
