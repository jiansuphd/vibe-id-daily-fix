# Day 72: Agentic 'Help Desk' Simulation

- **The Pointer:** Simulate a faculty help desk interaction to pre-build FAQ responses and identify the top 10 support gaps before a course launches.
- **The Details:** Faculty ask the same questions every semester. An agent that roleplay-simulates 20 common faculty questions against your course design identifies gaps before they become tickets.
- **Action:**

```bash
gemini "Simulate 10 faculty questions about this Canvas course. For each question, provide: the likely question, the correct answer, and the course element that should be clearer."
```
