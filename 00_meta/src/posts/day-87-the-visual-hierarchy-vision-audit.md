# Day 87: The 'Visual Hierarchy' Vision Audit

- **The Pointer:** Submit any course page screenshot to a vision model and receive a visual hierarchy audit with redesign recommendations.
- **The Details:** Visual hierarchy is invisible to the author but immediately felt by learners. A fresh AI 'eye' identifies where attention is being split, where CTAs are buried, and where whitespace is wasted.
- **Action:**

```bash
gemini --vision course_page.png "Audit the visual hierarchy of this course page. Identify: (1) first element the eye lands on, (2) elements competing for attention, (3) three specific layout improvements."
```
